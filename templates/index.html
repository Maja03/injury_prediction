Select a player from the list to view detailed injury risk analysis and interactive charts.

About this App

This dashboard estimates injury risk for professional football players using a trained machine learning model. It loads your processed dataset, makes predictions per player, and visualizes team-level risk and feature importance.

How to use
  • Use the search box to find a player by name.
  • Filter by position to narrow the list.
  • Click a player to view their predicted days injured, risk level, and explanation.
  • Review the Feature Importance chart to see which factors influenced the prediction.
Notes and limitations
  • Predictions are estimates; real outcomes can differ.
  • Actual Days shown uses total_days_injured when available, otherwise the latest season value.
  • Positions are mapped from numeric values in the dataset (0=Goalkeeper, 1=Defender, 2=Midfielder, 3=Forward).
  • The dataset does not include club information, so club filtering is not available.

Built for research and decision support. Always combine model outputs with expert judgment.
